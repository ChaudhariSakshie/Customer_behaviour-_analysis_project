# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project that analyzes customer shopping behavior using **Python, PostgreSQL, SQL, and Power BI**. This project uncovers customer purchasing patterns, product performance, subscription behavior, and revenue trends through data cleaning, exploratory data analysis (EDA), SQL querying, and interactive dashboard visualization.

---

## 📌 Project Overview

Retail businesses generate large amounts of customer data every day. The goal of this project is to transform raw transactional data into actionable business insights that support better decision-making.

Using a dataset of over **3,900 customer transactions**, this project explores customer demographics, purchasing behavior, discount usage, product performance, subscription trends, and customer segmentation to identify opportunities for business growth.

---

## 🎯 Objectives

- Analyze customer purchasing behavior
- Identify high-value customer segments
- Evaluate the impact of discounts on sales
- Understand subscription and shipping trends
- Discover top-performing products and categories
- Build an interactive business dashboard using Power BI

## 📊 Project Workflow

### 1. Data Cleaning & Preprocessing
- Imported the dataset into Python
- Handled missing values
- Renamed columns using snake_case
- Removed redundant columns
- Prepared the dataset for SQL analysis

### 2. Exploratory Data Analysis (EDA)
- Customer demographics analysis
- Product category analysis
- Revenue analysis
- Discount analysis
- Customer rating analysis
- Purchase frequency analysis

### 3. SQL Analysis
Performed SQL queries to answer business questions such as:
- Revenue by gender
- Revenue by age group
- Customer segmentation
- Product performance
- Subscription analysis
- Shipping preferences
- Discount impact

### 4. Power BI Dashboard
Developed an interactive dashboard featuring:
- KPI Cards
- Revenue Analysis
- Customer Segmentation
- Product Performance
- Subscription Insights
- Revenue by Category
- Revenue by Age Group
- Interactive Filters and Slicers

---

## 📈 Key Insights

- Male customers generated the highest overall revenue.
- Young adults contributed the largest share of sales.
- Non-subscribers accounted for most of the total revenue.
- Several products showed a high dependency on discounts.
- Loyal customers formed the largest customer segment.
- Subscription conversion represents a significant growth opportunity.
